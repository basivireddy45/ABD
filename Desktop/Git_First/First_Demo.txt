GitHub (/ˈɡɪthʌb/) is a developer platform that allows developers to create, store, manage and share their code.
 It uses Git software, providing the distributed version control of Git plus access control, bug tracking, software feature requests, task management, continuous integration, and wikis for every project.[6]
 Headquartered in California, it has been a subsidiary of Microsoft since 2018.[7]

It is commonly used to host open source software development projects.[8]
 As of January 2023, GitHub reported having over 100 million developers[9] and more than 420 million repositories.
